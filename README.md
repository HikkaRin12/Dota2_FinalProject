# 🎮 Dota 2 Final Project

Веб-приложение по Dota 2 с возможностью просмотра информации о героях. Проект создан как итоговая работа с использованием современных frontend-технологий.

## 🔗 Demo

Live version:

https://dota2-final-project.vercel.app/

---

## ✨ Возможности

- 🔍 Поиск героев Dota 2
- 📊 Просмотр статистики
- 🏆 Информация о матчах
- ⚔️ Детали героев
- 📈 Аналитика и игровые показатели
- 🌙 Адаптивный интерфейс
- ⚡ Быстрая загрузка данных через API

---

## 🛠️ Технологии

Проект построен с использованием:
-CSS
-JavaScript
-HHTML
- Vercel Deployment

## 📦 Установка проекта

Клонировать репозиторий:

```bash
git clone <repository-url>
```

Перейти в папку проекта:

```bash
cd project-name
```

Установить зависимости:

```bash
npm install
```

Запустить локально:

```bash
npm run dev
```


## 🚀 Production build

Создать production сборку:

```bash
npm run build
```

Запуск:

```bash
npm start
```
